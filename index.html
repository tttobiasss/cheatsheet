Cheatsheet voor handige dingen

Position property (css)

Property Naam Beschrijving
Position: static; Static De standaard, staat gewoon op volgorde van de html
Position: relative; Relative Voor andere elements op de normale plek, maar verplaatsbaar met top:, left:, etc.
Position: absolute; Absolute Negeert "siblings" geplaatst relatief naar de "parent" element zit niet in flow van doc.
Position: fixed; Fixed Blijft op dezelfde plek, ook als je scrollt gebruikt voor navbar zit niet in flow van doc.

<Table> tags

Tag Naam Beschrijving
<table> Table De tags die om de hele tabel heen gaan.
<thead> Table Head Daar gaan de Table Rows voor de headers van de kolom in.
<tbody> Table Body Daar gaan de Table Rows voor de data van de table in.
<tr> Table Row De tags voor 1 rij van de tabel
<td> Table Data Blijft op dezelfde plek, ook als je scrollt gebruikt voor navbar zit niet in flow van doc.
<tfoot> Table Foot Blijft op dezelfde plek, ook als je scrollt gebruikt voor navbar zit niet in flow van doc.
